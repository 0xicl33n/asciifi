ASCII art has been around since the 60s, but we're still using crappy PHP scripts to upload and transform images.

There has to be a better approach, right? Imagine an ASCII art converter for a more civilized age.

Want to drag a photo onto the page? Go ahead. Want to upload from your computer or link to an external image? Have at it.

Note There is one catch. You will have to use a good browser — one like Google Chrome or Firefox. Safari doesn't support the FileReader API, and, sadly, if you're using an old version of IE, then you've come to the wrong place.
